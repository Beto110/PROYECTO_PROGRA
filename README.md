# Sistema Web de Control de Inventario y Ventas (InvenWeb)
##  Descripción
InvenWeb es un sistema web diseñado para optimizar y automatizar el control de inventario y el registro de ventas. Proporciona una interfaz intuitiva 
con operaciones completas (CRUD) para productos y categorías, un módulo avanzado de carrito de compras, cálculo de existencias en tiempo real y protección mediante
roles de usuario.

##  Base de Datos
El proyecto utiliza **SQLite** como Sistema Gestor de Base de Datos, lo que permite un despliegue rápido, ligero y embebido en el mismo servidor. El archivo de inicialización de la base de datos se encuentra dentro de la carpeta `backend/` (`init_db.php` que genera `database.sqlite`).
